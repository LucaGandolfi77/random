### **Quick Checklist for Future Drafts**

1. **Keep exposition embedded in action.** Let the setting emerge through what characters do, not through a narrator’s lecture.  
2. **Trim dialogue tags** that repeat the same beat (“she said,” “she whispered”). Use small physical actions to carry subtext.  
3. **Make stakes explicit early.** In a cyber‑punk noir, the reader needs to know exactly what’s on the line (money, identity, freedom).  
4. **Give each character a distinct voice.** Vesper’s dialogue should feel tighter and more controlled than Maren’s gritty, desperate tone.  
5. **End on a visual hook.** The drone’s lingering gaze and the child’s echo are perfect bookends—keep them crisp and uncluttered.
