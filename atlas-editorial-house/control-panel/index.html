    Telegram Control
    
          
    The Telegram bridge polls the Bot API locally and forwards commands to this server. It does not expose a webhook or public HTTP endpoint.
    
        
    
        
    
          
    Core Telegram commands
    
          
      
            
    • /scenarios lists available launcher presets.
      • 
            
    • /agents lists available personalities.
      • 
            
    • /preview article previews a scenario.
      • 
            
    • /run article launches a scenario job.
      • 
            
    • /custom Use the atlas-editorial-house skill... runs a custom prompt.
      • 
            
    • /status JOB_ID shows status and log tails.
      •
