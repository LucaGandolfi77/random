# HERE IS TOO HARD TO SMILE
## Project Work Order

### Team Structure
- **Director**: Orchestrates workflow, ensures continuity, manages review process
- **Literary Editor**: Oversees manuscript coherence, enforces 3000-word minimum
- **Critic**: Provides specific risk analysis per chapter
- **Finisher**: Ensures terminology consistency, removes padding
- **Translators**: Handle English-to-Italian translation with cultural nuance

### Definition of Done
1. All chapters meet 3000-word minimum
2. Critical risks identified and documented per chapter
3. Terminology normalized across all chapters
4. All required files created (see 99_manifest.md)
5. Italian translations complete with translation notes

### Workflow
1. 01-03 planning documents completed first
2. Chapter drafting follows sequential order (01-20)
3. Chapter 01 must complete before chapter 02 begins
4. Parallel translation work starts after English completion
5. Final verification via 99_manifest.md
