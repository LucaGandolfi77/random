# Red Inheritance: Work Order

## Project Brief

- **Title**: Red Inheritance
- **Theme**: Political power, family betrayal, institutional decline
- **Genre**: Literary novel
- **Chapters**: 20
- **Minimum words per chapter (English)**: 3000
- **Translation required**: Yes (to Italian)
- **Output directory**: local-output/books/red-inheritance/

## Team Assembly

| Role | Agent | Responsibility |
|------|-------|----------------|
| Lead | tolstoy | Structure, architecture, first-pass chapter drafts |
| Reviewer | austen | Quality check against brief, pacing, motive, consistency |
| Critic | poe | Stress-test assumptions, melodrama, drift, weak resolutions |
| Refiner | woolf | Transitions, tonal flow, interior continuity |
| Finisher | hemingway | Compression, terminology normalization, sign-off |

## Route

1. **Lead (tolstoy)** drafts each chapter to 3000+ words
2. **Reviewer (austen)** checks quality against brief and character bible
3. **Critic (poe)** stress-tests for melodrama, drift, and unearned resolution
4. **Refiner (woolf)** improves transitions and interior continuity
5. **Finisher (hemingway)** normalizes terminology and approves shipment

## Definition of Done

- [ ] All 20 English chapters exist with 3000+ words each
- [ ] All chapters pass critic review with named risk mitigation
- [ ] All chapters pass finisher sign-off (compressed, normalized)
- [ ] All 20 Italian translations exist in `it/` subdirectory
- [ ] `98_translation_notes.md` documents key translation decisions
- [ ] `99_manifest.md` records completion status for every file
- [ ] `03_character_bible.md` updated for any name/institution changes
- [ ] Book feels like one coherent novel, not disconnected episodes

## Primary Risks

1. **Institutional drift**: Political machinery may become generic rather than specific
2. **Family melodrama**: Betrayal may tip into soap opera without grounding
3. **Voice fatigue**: 60,000+ words may strain a single narrative approach
4. **Temporal clarity**: Multiple timelines need clear anchoring

## Handoff Sequence

```
Lead → Reviewer → Critic → Refiner → Finisher → Archive
```

Each chapter follows this sequence before the next begins. The `03_character_bible.md` is updated after each chapter if continuity requires changes.
